# Storybook for ProView Demo
Storybook is a frontend workshop for building UI components and pages in isolation.
## Key points
1. Rapid development of reusable components
2. [Automated testing](https://storybook.js.org/docs/react/writing-tests/introduction) (a11y, unit, end-to-end)
3. Optimized team workflow
    1. PR triggers visual review by everyone ([Chromatic](https://www.chromatic.com/))
    2. Integrates with existing tools (Figma, Jira, Github/Azure DevOps and more)
4. Publishable as an NPM package
    1. Quick installs & releases
5. Built-in [documentation](https://storybook.js.org/docs/react/writing-docs/introduction)
